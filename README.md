## ✨ Features

- Upload `.txt` or `.pdf` files for word-by-word reading  
- **Adaptive Reading Engine**: automatically adjusts speed based on word length and punctuation  
- **Focus Mode**: hides all UI elements for distraction-free reading  
- Click any word in the text to start reading from that point  
- Highlight selected text for targeted reading  
- Customizable **Words Per Minute (WPM)** via slider or manual input  
- Multiple themes  
- Keyboard controls for Play / Pause / Reset  
- Interactive progress bar and word highlighting
