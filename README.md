[![Made with React](https://img.shields.io/badge/Made%20with-React-blue)](https://reactjs.org/)
[![Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-00c7b7)](https://vercel.com/)

**VeloxiLector** is a modern, web-based speed reading app that helps users read text and PDFs efficiently, with features like focus mode, adaptive speed, and interactive word selection.

---

## ✨ Features

- Upload `.txt` or `.pdf` files for word-by-word reading  
- **Adaptive Reading Engine**: automatically adjusts speed based on word length and punctuation  
- **Focus Mode**: hides all UI elements for distraction-free reading  
- Click any word in the text to start reading from that point  
- Highlight selected text for targeted reading  
- Customizable **Words Per Minute (WPM)** via slider or manual input  
- Multiple themes  
- Keyboard controls for Play / Pause / Reset  
- Interactive progress bar and word highlighting  

---

## 💻 Technologies Used

- **React.js** (frontend)  
- **pdfjs-dist** (PDF parsing)  
- **CSS / Custom Styling**  
- **Vercel** (deployment)
